Key features

Any shape solver

Solve rectangular, trapezoidal, stepped or fully custom polygons that you draw on the canvas. The shaded compression region drives the math.

Custom rebar layers

CSA metric bar sizes, bundles and side cover. Define exact distance from top for multiple layers with area weighted depth.

CSA A23.3 analysis

Supports CSA A23.3 using α1 and β1 with an iterative equilibrium search for accuracy in a and c. Mn uses lever arms from each steel layer. Coming soon: AISC, AASHTO, and nominal capacity solvers.

Interactive visuals

Live compression block shading, lever arm display and per layer force readouts keep the flow transparent.

Clean exports

Export PNG snapshots ready for reports, teaching and quick reviews.

Compact UI

Focused inputs and tidy panels with a red and gray theme keep everything readable without clutter.
